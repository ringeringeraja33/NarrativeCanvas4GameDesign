# Narrative Canvas for Game Design

Narrative Canvas is a node-based story planning canvas for game design. It can run as an Obsidian community plugin and also opens as a standalone local HTML page.

## Install in Obsidian

1. Copy this repository folder into your vault at:

   ```text
   .obsidian/plugins/narrative-canvas
   ```

2. Restart Obsidian, or reload community plugins.
3. Enable `Narrative Canvas` in `Settings -> Community plugins`.
4. Open it from the ribbon icon or the command palette command `Open Narrative Canvas`.

## Files

- `manifest.json`: Obsidian plugin manifest.
- `main.js`: Obsidian plugin entry point.
- `styles.css`: Obsidian host view styles.
- `index.html`: Canvas UI shell.
- `app.js`: Canvas interaction logic.
- `canvas.css`: Canvas UI styles.

## Standalone Preview

Open `index.html` in a browser to preview the canvas outside Obsidian.
